Your archive does not have the program that I am looking for. Where can I find more TI 83/83+ programs?
It is quite possible that the growing archive of BASIC Guru Online does not have the program that you're looking for. If this is the case, then you need to try the other major TI calculator archives. The big one (which is probably the best one) is ticalc.org. Coming is section is Dimension TI. The two archives basically have the same programs, but ticalc.org has better web design. You can also try out TI-Files, Twilight-TI, or even TI itself. There are many other TI calculator sites that have program archives including some listed in the Links Page. If it so happens that you still do not find the type of program that you're looking for in all the sites that I've given you (this is highly unlikely), then you should go to Yahoo! and search for TI archives. If you still do not find it (that would pretty much amazing), then you should think about making it yourself because you're obviously the only one who has this idea for a program. If you need any help making it, just send me what you need help in, through the Questions form.
