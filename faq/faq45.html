Which emulator should I use to have my TI-83/TI-83 Plus on my computer?
I recommend using Rusty Wagner's Virtual TI emulator. I used it and it worked perfect for me without any flaws. The really good thing about his emulator is that it works for Windows and for all types of calculators. So if you have more than one calculator that you want to emulate, this is the emulator to use. An emulator is useless without a ROM (Read Only Memory) image, which contains the software you use to operate the calculator. Another good thing about the Virtual TI is that it has a step-by-step process for downloading the ROM image from your calculator. Obviously, Rusty's emulator isn't the only one; there are others too. You can check out a list of many emulators on a page at ticalc.org.
