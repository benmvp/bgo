How do I Archive/Unarchive inside a program?
Inside a program, you may archive a variable by first pressing and [MEM] and then choosing 5:Archive. Type in the variable name immediatly after "Archive". The variable can be a string, function, matrix, user variable, program, or anything else that can be archived in the memory menu. If you want to archive a program, you would have to have "prgm" before the program name. For example, if you wanted to archive a program called JUMP, you would have a command saying "Archive prgmJUMP". If you wanted to archive user variable K, you would have a command saying "Archive K". This process works the same for UnArchive. If you archive a variable that is already archived (or unarchive a variable that is already unarchived), the command will be ignored. Arhicve and Unarchive do not work on the TI-83.
