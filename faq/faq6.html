Is there a place where I can interact with the other users of BASIC Guru Online?
Of course there is! You can interact with the other users of BASIC Guru Online by using the Message Board. The Message Board allows you to reply to already posted messages or post your own. Whenever somebody replies to a message that you posted, you will receive an email with what they had to say. This can also be another arena for you to asks questions except for the fact that everybody gets to see your question. This way, it can be a group effort to solve your question.

You can also use the BASIC Guru Online Chatroom to interact with other visitors of the site. It is open 24 hours a day, but I will most likely be in it only during the evenings. For times, go to the Interact page.

Do you need some ideas for a program that you want to create? Do you have some ideas for a program, but just don't know how to create it? Either way, you can post a request for ideas for a program you're making or post some ideas that you have for a program on the BASIC Guru Online Program Idea Message Board.
