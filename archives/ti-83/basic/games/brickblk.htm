1.0 ABOUT Brick Block
1.1 What is Brick Block?

Brick Block is a very innovative program, made by SiCoDe. It's engine is based on a year of development. It is a brand new puzzle game that the TI-Basic community has never seen before.

Please note that the Brick Block Level Editor is included in this ZIP, but has a separate manual (BRICKEDT.HTM).

2.0 INSTALLATION INSTRUCTIONS
1) Unzip the file BRICKBLK.ZIP into any directory, usually your Link Software's directory (e.g. C:\WINLINK).
2) Run your Link Software and send the file BRICKBLK.83G to your TI-83/83+.
3) Run the program prgmBRICKBLK to begin.
4) Visit SiCoDe's website at http://sicode.ticalc.org/ for news and program updates.

4.0 Instructions
Objective-Advance your figure(little +) to the big plus sign by pushing and pulling the white moveable blocks out of the way. Push blocks by running into them, use assigned key to pull them.
Other blocks- Teleports-The small x is the one that warps you, the x with dots around the corners is where you go.Don't forget that teleports are only one way and that you can pull blocks.
4.1 Scoring
  • Scoring is based on two things. 1)How efficient you are,ie. time vs work done. 2)How well you solve the puzzle,ie. your number of moves vs. lowest possible number of moves. The two parts are averaged together to get an integer score from 0-10.

SiCoDe is a group of programmers who program for various TI calculators in the TI-BASIC language. SiCoDe aim to continually provide high-quality TI-BASIC programs. These are frequently considered to be inferior in all ways to ASM programs and we aim to eliminate this misconception!
SiCoDe is an affiliate of the TI-BASIC Quality Alliance which means that quality is assured in our products.
