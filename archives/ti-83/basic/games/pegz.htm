1.
 What Is It?
Pegz is a simple puzzle game that I found off the web and decided to make into a calculator game.

2.
 Installation:
1. Unzip the file PEGZ.ZIP into any directory, usually your Link Software's directory (e.g. C:\WINLINK).
2. Run your Link Software and send the file PEGZ.83G to your TI-83/83+.
3. Run the program prgmPEGZ to begin.
4. Visit SiCoDe's website at http://sicode.ticalc.org/ for news and program updates.

3.
 Game Controls:
KEY
ACTION
Window,Zoom,Trace Select a peg to move disc from, push again on different peg to move that disc.
Clear Quit current game
Any other key Deselects a selected peg

4.
 Objective:
The point of Pegz is to move all the discs from the beginning peg to a different peg. The catch is that you can only place smaller discs on top of larger discs. There is a possible solution, some may find the challenge harder than others.

Scoring:
Scoring is very simple, the number of moves if you complete the puzzle, is your score.

5.
 Troubleshooting:
I get a memory full error: Reset your calculator then reload Pegz.
I don't know how to transfer the program to my calc: Read the instructions that came with your link software.

6.
 About SiCoDe:
SiCoDe is a group of programmers who program for various TI calculators in the TI-BASIC language. SiCoDe aim to continually provide high-quality TI-BASIC programs. These are frequently considered to be inferior in all ways to ASM programs and we aim to eliminate this misconception!

SiCoDe is an affiliate of the TI-BASIC Quality Alliance which means that quality is assured in our products.
