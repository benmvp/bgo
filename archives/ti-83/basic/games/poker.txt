POKER v2.0
Black Software

Ti83 basic game

The game is pretty simple:

	<Graph keys> to hold or to remove the hold the card
	<2nd> Change: Remove -> Hold / Hold -> Remove

It uses lists and only takes 3802 bytes.
I don't think there are any bugs in it.
