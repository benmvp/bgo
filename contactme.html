Frequently Asked Questions (FAQ)
This should be your first stop if you have a question. Check the FAQ to see if your question has already been answered. Go here to find answers to questions about this site, the TI-83, the TI-83 Plus, linking, general usage, and more.
Site Search
Know what you're looking for, but just don't know where to find it? Go to the Site Search page to search through BASIC Guru Online. There is a pretty big chance that your question is answered somewhere on the site.
General Message Board
Do you have a question that you feel that a lot of others might have also? You can post your question in the BASIC Guru Online General Message Board. Questions posted on the General Message Board usually are replied to within 6-12 hours depending on when you post it.
Comments/Questions/Suggestions
Go here if you have a commment, question, or suggestion about a specific tutorial, the overall site, etc. This is the place to ask for help. If you send a question that's already answered in the FAQ, the reply will tell you to go to the FAQ to find the answer to your question.
TI-83 Guidebook
This is the online version of the book that came with your TI-83 that serves as the manual for your TI-83. You would be suprised at how well it can help you understand how a command works. Chapter 16 is the chapter that covers the programming commands. If you just want to figure out how to use a command or if it can do a certain task, use the Guidebook. It'll most likely answer your question. You will need Adobe® Acrobat® Reader™ to view the files. Don't worry, it is free.

Tutorial Code Problems/Errors
Go here if you have possibly found an error in one of the codes in one of the tutorials. Please include what tutorial you found the error, which set of code the error is located, and the line where you received an error on your calculator. Including all of this information will make it easier to determine if in fact there is an error in that particular code.
